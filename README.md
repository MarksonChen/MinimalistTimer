# MinimalistTimer

This is a minimalist stopwatch that floats on your screen, and can be started/paused/resetted just with a click!

Download [MinimalistTimer.jar](https://github.com/MarksonChen/MinimalistTimer/raw/master/MinimalistTimer.jar) and try it out!

**_Left Click_** - Start / Pause the stopwatch

<img src="https://github.com/MarksonChen/MinimalistTimer/assets/46666959/c155ea17-e5ab-4a09-9857-fb35bd56967f" width="30%" height="30%"/>

**_Right Click_** - Reset the stopwatch

<img src="https://github.com/MarksonChen/MinimalistTimer/assets/46666959/a4a600ce-5bb0-476e-a66a-79938a625e98" width="30%" height="30%"/>

**_Left Mouse Drag_** - Drag the stopwatch around

<img src="https://github.com/MarksonChen/MinimalistTimer/assets/46666959/1e820c91-81a7-4406-9a51-2ba16684aade" width="30%" height="30%"/>

**_Mouse Scroll_** - Make the stopwatch larger/smaller
